Sobre o Projeto

Este projeto foi desenvolvido utilizando Flask e Bootstrap. Abaixo estão os detalhes das tecnologias usadas:

  • Flask: Um framework web em Python usado para construir a aplicação.
  • Bootstrap: Um framework CSS para estilização e design responsivo da aplicação.
  • SQLite: Banco de dados leve usado para armazenar itens no projeto.
  • Werkzeug: Biblioteca para segurança e autenticação.
